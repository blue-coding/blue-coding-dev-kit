# Blue Coding — Marketing UI Kit

High-fidelity component recreation for the Blue Coding marketing website surface.

## Screens
1. **Homepage hero** — headline, CTA, image
2. **Services section** — 3-column feature cards
3. **Social proof / stats** — numbers bar
4. **Team card grid** — engineer profile cards
5. **Contact / CTA section** — dark closing strip

## Usage
Open `index.html` for the interactive prototype.
Import individual JSX components into any design artifact.

## Components
- `NavBar.jsx` — top navigation with logo + links + CTA button
- `HeroSection.jsx` — display headline, divider, subtitle, CTAs, image
- `ServicesSection.jsx` — 3-up icon cards with titles + descriptions
- `StatsBar.jsx` — horizontal stats strip (dark background)
- `TeamCard.jsx` — engineer profile card with photo placeholder
- `TestimonialCard.jsx` — quote card with attribution
- `CTASection.jsx` — dark closing strip with gradient headline + buttons
- `Footer.jsx` — site footer with logo, links, social icons
